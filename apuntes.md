### CSS
 "#" es el selector del identificador (id)

# Tamaños
- width = ancho
- length = largo
- max-width = Fija un ancho máximo de X píxeles para el contenedor. (El elemento que haya dentro no crecerá más de X).

# Márgenes
- margin
- margin:auto = Aplica márgenes automáticos en los cuatro lados. Lo más improtante es que centra horizontalmente un bloque cuando su ancho está limitado.
# Relleno
Padding = Añade espacio interno de X píxeles en los cuatro lados del elemento
